# Swiggy Sales Analytics Dashboard

## Project Overview
This project analyzes Swiggy sales data using Python, SQL, and Power BI to generate business insights and interactive dashboards.

## Tools Used
- Python
- SQL
- Power BI
- Excel

## Features
- Revenue Analysis
- City-wise Sales Performance
- Category Analysis
- Revenue by Location Type
- KPI Dashboard
- Yearly Revenue Trends

## Workflow
1. Data Collection
2. Data Cleaning using Python
3. Data Analysis using SQL
4. Dashboard Creation in Power BI
